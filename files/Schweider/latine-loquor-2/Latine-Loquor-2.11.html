Persōnae:
Mīchael, Norbertus.
Mīchael
Vīsne mihi commodāre tuum Horātium?
Norbertus
Volō equidem, modo illum repetās ā Conrādō, cui ūtendum dedī.
Mīchael
Quō sīgnō vīs repetam?
Norbertus
Nempe hoc, quod eius habeō epistulās.
Mīchael
Id mihi satis est.
Norbertus
Sed quandō reddēs?
Mīchael
Cum dēscrīpserō contextum in trēs aut quattuor praelēctiōnēs.
Norbertus
Mātūrā igitur: nē meō studiō incommodēs.
Mīchael
Mātūrābō.
Norbertus
Sed heus tū, cavē maculēs: aliōquīn aegrē commodābō posthāc.
Mīchael
Nempe indignus essem beneficiō.

Ēlocūtiō

  • Commodāre alicuī aliquid ... ūtendum dare.
  • Dare commodātō alicuī aliquid.
  • Commodā mihi Līvium tuum, tantisper dum lūdis.
  • Repetere rem aliquam ... repetere locum aliquem.
  • Dare alicuī aliquid ūtendum ... dare alicuī aliquid dēscrībendum.
  • Dare alicuī aliquid dīscendum.
  • Dedit mihi hanc epistulam dēscrībendam.
  • Hoc sīgnō volō repetās.
  • Satis est mihi ... sat diū ... sat est ... sat sciō ... satis verbōrum est.
  • Quandō reddēs mūtuum? ... reddere librum ... reddere litterās alicuī.
  • Reddere ratiōnem ... reddere respōnsum ... reddere alvum.
  • Dēscrībere aliquid ... dēscrībere aliquem.
  • Dēscrībere aliquem fūrem.
  • Mātūrandum est ... mātūrātō opus est ... mātūrāre abitum ... mātūrāre fugam ... mātūrāre iter.
  • Incommodāre alicuī ... incommodāre studiīs alicuius.
  • Inicere remorāmen studiīs alicuius ... impedīre studia alicuius.
  • Cavē illum ... cavē tibi ab illō ... cavē ab illō.
  • Ille tibi vītandus est. Tibi est ab illō cavendum.
  • Sī sapīs, illum fugiēs.
  • Cavē improbōs sodālēs ... Fuge prāva cōnsortia.
  • Vītā malōrum cōnsuētūdinem.
  • Liber meus mundior tuō ... amō munditiem librōrum.
  • Cavē, maculēs meum librum digitīs ātrāmentō foedātīs.
  • Habet aurēs vel auriculās asīnōrum.
  • Cavē nē fallāris, vel cavē fallāris.
  • Librī forīs et intus commaculātī.
  • Indignus beneficiō.
  • Plūrimī Chrīstiānī beneficiīs Deī indignī sunt.
